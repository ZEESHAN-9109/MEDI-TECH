# 🏥 MediTech - AI Powered Healthcare Management Platform

MediTech is a modern healthcare management platform designed to improve healthcare accessibility for individuals and families. It combines AI-powered medical assistance with health tracking, medication management, appointment scheduling, and community health support in a single web application.

The platform is designed with a mobile-first approach and focuses on providing an easy-to-use healthcare solution for both urban and rural communities.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/meditech.git
```

Navigate into the project

```bash
cd meditech
```

Install dependencies

```bash
npm install
```

Create a `.env` file and add:

```env
DATABASE_URL=your_database_url
OPENAI_API_KEY=your_openai_api_key
```

Start the development server

```bash
npm run dev
```
